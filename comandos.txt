-> python manage.py migrate:
aplica os scripts de banco de dados (cria as tabelas)

-> python manage.py createsuperuser:
adiciona um usuario administrador (admin)
ex: admin - admin

-> python manage.py createuser:
adiciona um usuário comum (sem ser staff)
